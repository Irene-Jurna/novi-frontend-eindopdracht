# Vegan recepten webapplicatie

## Inhoud

Vegetarisch koken, daar hebben de meeste mensen inmiddels wel een paar recepten voor op de plank liggen. Maar een groter bewustzijn met betrekking tot klimaat en gezondheid leidt bij een steeds meer mensen tot nieuwe eetwensen (denk aan FODMAP, lactosevrij en veganistisch). In deze webapplicatie kun je recepten zoeken die passen bij die verschillende eetwensen.

## Project opstarten

Download deze Git-repository. Maak een gratis account aan bij [Edamam API](https://developer.edamam.com/edamam-recipe-api). Als je je registreert, krijg je een ID en een key. Vul jouw persoonlijke ID en key in in dit project in jouw favoriete IDE, achter de variabelen `edamameId` en een `edamameKey`. Deze vind je op de pagina's: home, recipe questions, en recipe search function.

Dat is alles wat je hoeft te doen om het project te runnen.

Veel (kook)plezier!
